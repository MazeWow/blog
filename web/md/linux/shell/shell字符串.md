字符串长度
================================================================================
```bash
➜  shell git:(master) var=122131213122sfsfsdfd
➜  shell git:(master) echo ${#var}
20
```

字符串比较
================================================================================
- `[[ $a==$b ]]`相等
- `[[ $a!=$b ]]`不等
- `[[ $a>$b ]]`字符序大
- `[[ $a>$b ]]`字符序小
- `[[ -z $a ]]`空字符串
- `[[ -n $a ]]`非空字符串
- `if [[ -z $a ]] && [[ -n $b ]]`将多个条件组合起来

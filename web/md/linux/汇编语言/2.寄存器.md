# 汇编指令
```
mov ax,18  // AX = 18
mov ah,78  // AH = 78
add ax,8   // AX = AX + 8
mov ax,bx  // AX = BX
add ax,bx  // AX = AX + BX
```

# 8086如何计算出20位地址
- CS : 段寄存器
- IP : 偏移寄存器，读取一条指令后，IP的值会自动增加，比如当前读入的指令为3个字节，所以IP自增3,CS:IP 指向 2000:0003

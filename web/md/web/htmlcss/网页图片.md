# 背景图
```css
border-image:url(borderimg.png) 70 repeat; /* 切割图片，生成边框的！*/
background-color:rgba(100,120,60,0.5); /* 颜色透明度 */
background-image:linear-gradient(45deg,white,black,red); /* 线性渐变 左下角到右上角的线性渐变*/
```

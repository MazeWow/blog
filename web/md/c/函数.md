# 函数
- 函数被翻译为一段相对独立的汇编代码,函数名作为标号，表示此段代码的入口，它的值等于内存中存储函数代码的内存空间首地址
- 调用函数就是跳转到这个标号，执行标号后面的指令
- C语言中不允许作嵌套的函数定义。因此各函数之间是平行的，不存在上一级函数和下一级函数的问题。所以，函数可以调用自身，也可以两个函数相互调用，但是一定要有退出条件，不能无限循环调用下去！

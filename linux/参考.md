# Clang
得益于Apple的Clang。
注：Clang 是一个 C++ 编写、基于 LLVM、发布于 LLVM BSD 许可证下的 C/C++/Objective C/Objective C++ 编译器，
其目标（之一）就是超越 GCC。由于使用了Clang所以这个插件在离线安装模式下（因为开发机没有网络，或者Proxy不太好用）
就复杂一点，需要先安装最新版本的GCC才能编译Clang。
